## 📖 About

**ГМЦ (Городской Медицинский Центр)** is a modern medical center website designed to provide patients with easy access to medical services, doctor information, promotions, and online appointment booking. The site is fully responsive and works seamlessly across all devices.

## ✨ Features

- 🎨 **Modern & Clean Design** - Professional medical aesthetic
- 📱 **Fully Responsive** - Works on mobile, tablet, and desktop
- 🏥 **Multiple Service Pages** - General medicine, pediatrics, cosmetology
- 👨‍⚕️ **Doctor Profiles** - Meet the medical staff
- 📅 **Online Appointment Booking** - Easy booking forms
- 💬 **Patient Reviews** - Testimonials section
- 🎁 **Promotions & Discounts** - Special offers section
- 📞 **Contact Integration** - WhatsApp, phone, and email
- 🗺️ **Interactive Map** - Location and directions
- 📜 **Licenses & Certificates** - Trust indicators
- ♿ **Accessibility** - Version for visually impaired users

## 📄 Pages

1. **Home Page** (`index.html`) - Main landing page with services overview
2. **Pediatrics** (`children.html`) - Children's medical services
3. **Cosmetology** (`cosmetology.html`) - Cosmetology services
4. **Cosmetology Service** (`service.html`) - Detailed service page
5. **Contour Plastic** (`contour.html`) - Specific procedure page

## 🛠️ Technologies Used

- **HTML5** - Semantic markup
- **Tailwind CSS** - Utility-first CSS framework
- **JavaScript** - Interactive functionality
- **Font Awesome** - Icons
- **Google Fonts** - Typography
