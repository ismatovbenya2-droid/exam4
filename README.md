# 🏥 Medical Center Website

A modern, responsive medical center website built with **HTML5** and **Tailwind CSS**.

## 📖 About

This project is a responsive multi-section landing page for a medical center. It includes service listings, doctor profiles, articles, testimonials, certificates, pricing, contacts, and other healthcare-related content.

The layout is fully responsive and optimized for desktop, tablet, and mobile devices.
